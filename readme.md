Suchen und Zerstören
=============

Klaus Bleck, 466740
Jülide Meyer, 455320

Suchen und Zerstören ist ein Multiplayer-VR-Game für Google Cardboard. Getestet für Android.


Aufbau des Projektes
------------------
* Assets/Cardboard/* - Google Cardboard SDK für Unity, eigene Anpassungen sind in einer Datei: Scripts/GazeInputModule.cs
* Assets/Models/* - alle verwendeten 3D-Modelle (cage, colors und shelf sind Demo-Modelle von MagicaVoxel, die restlichen Modelle sind selbst erstellt)
* Assets/Prefabs/* - angelegte Prefabs für die 3D-Modelle
* Assets/Scenes/Main.unity - Hauptszene
* Assets/Scripts/* - alle Skripte
* Assets/SimpleSky - externes Unity-Package für die Skybox
* Assets/Sounds - alle Sounds (sind extern von soundbible.com)
* Assets/Texturen - eigene Texturen für das Terrain
* Assets/UI/Buttons/* - alle 2D-Elemente für die UI, Grafiken wurden selbst erstellt bzw. angepasst, allerdings mit einer Vorlage von http://graphicburger.com/mobile-game-gui/



Starten des Spiels
-----------------
1. Starten des Spiels im Unity Editor (ggf. gibt es Probleme mit der Windows Firewall)
oder
2. Starten des Spiels auf Android (APK liegt in ARGame-Root-Verzeichnis: Suchen_und_Zerstoeren_VR_Game.apk
